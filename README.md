Description: Design a RESTful system. There are respectively four features:

* get /user/:id
* post /user
* get /user
* update user/:id

### Test environment

* Ubuntu 14.04.5 LTS
* Python 3.4.3

### Get started

```bash
(Optional)
$ cd ezprice
$ virtualenv --no-site-packages -p python3 env
$ source env/bin/activate
=========================================
(env)$ pip install -r requirements.txt
(env)$ python ezprice.py
```
